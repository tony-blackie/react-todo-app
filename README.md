# Test application

Boilerplate for react/redux app.

## Technologies/Approaches

ES2015, babel, react, redux, less, bem, webpack, mocha, eslint, git

## Requirements

* node `^6.4.0`
* npm `^3.10.3`

## Getting Started

### Install from source

```bash
$ git clone <url> <my-project-name>
$ cd <my-project-name>
$ npm i
$ npm start
```
